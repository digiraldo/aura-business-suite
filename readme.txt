Suite modular de gestion empresarial para WordPress con permisos granulares (CBAC) e integracion entre modulos.

== Description ==

Aura Business Suite centraliza la gestion operativa de una organizacion en un solo plugin modular.

Incluye:

* Finanzas
* Inventario
* Estudiantes
* Certificados
* Formularios y Encuestas
* Vehiculos
* Biblioteca
* Electricidad
* Areas y Programas

Caracteristicas clave:

* Permisos granulares por capability (CBAC).
* Integracion entre modulos para flujos de trabajo completos.
* Reportes y exportaciones por modulo.
* Estandar de tablas responsive para admin.
* Soporte de notificaciones por email y WhatsApp segun configuracion.

== Installation ==

1. Sube la carpeta `aura-business-suite` al directorio `/wp-content/plugins/`, o instala el plugin desde el panel de WordPress.
2. Activa el plugin desde el menu Plugins en WordPress.
3. Abre Aura Suite en el panel de administracion.
4. Configura ajustes globales y permisos CBAC antes de iniciar operacion.

== Frequently Asked Questions ==

= El plugin es modular? =

Si. Cada dominio funcional se organiza en modulos con integraciones comunes.

= Soporta permisos granulares por usuario? =

Si. Usa un modelo CBAC con capabilities por modulo y accion.

= Se pueden generar reportes y exportaciones? =

Si. Los modulos principales incluyen reportes y exportacion segun permisos configurados.

= Es apto para produccion? =

Si, siempre que se valide configuracion, permisos y flujos criticos en tu entorno antes del despliegue final.
